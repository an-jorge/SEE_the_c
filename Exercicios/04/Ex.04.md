# Ex.04 par ou Impar

Preparar um algoritmo que leia um número inteiro e descubra se é par ou
ímpar.

```c
#include <stdio.h>

int main() {
    int number = 13;
    if (number % 2 == 0)
    {
        printf("Numero par");
    }
    else
    {
        printf("Numero impar");
    }
    return 0;
}
```
